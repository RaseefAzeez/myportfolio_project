My Portfolio Website

This is a Static website hosted on AWS S3 with a contact form connected via API Gateway, Lambda, and DynamoDB. Integrated email notification with SNS and CI/CD via GitHub Actions.

Architecture Overview

This diagram illustrates the serverless architecture of my portfolio website, featuring static hosting on Amazon S3, a contact form powered by AWS API Gateway and Lambda, data storage in DynamoDB, and real-time notifications via SNS. The entire infrastructure is managed as code using CloudFormation, with CI/CD automation handled by GitHub Actions for seamless deployments.


Technologies Used

  • AWS Services: S3 (Static Hosting), API Gateway, Lambda, DynamoDB, SNS, CloudFormation, IAM
  • CI/CD: GitHub Actions
  • Version Control: Git, GitHub
  • Frontend: HTML, CSS, JavaScript

Features

  • Static website hosting with high availability.
  • Serverless contact form with data persistence.
  • Real-time email notifications on form submission.
  • Infrastructure as Code (IaC) for consistent deployments.
  • Automated CI/CD pipeline for both frontend and backend.
